# honeytiper
##### 하이미디어 파이널프로젝트
### 꿀TIPer
꿀TIPer-쪽지를 통한 빠른 커뮤니케이션, 안전한 에스크로 결제, 당도(평점)를 통한 강의 선별, 간편한 강사등록으로 꿀팁을 전수하고 전수받는 플랫폼입니다
[꿀TIPer 보러가기](http://www.ggultiper.kro.kr)
